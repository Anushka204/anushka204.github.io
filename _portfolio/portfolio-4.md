## Project Description

This project focuses on analyzing credit card transaction data to identify customer spending patterns, usage trends, and potential risk indicators. Raw datasets were cleaned and transformed using Python data analysis libraries before visualization.

Interactive dashboards were built using Power BI to provide decision-makers with clear insights into customer behavior and financial trends.

## Key Features
- Data cleaning and preprocessing using Python
- KPI-based dashboard visualization
- Spending pattern and risk indicator analysis
- Interactive and user-friendly analytics interface

## Technologies Used
Power BI, Python, Pandas, NumPy, Data Visualization

## Impact & Learning
- Strengthened data analytics and visualization skills
- Learned data cleaning and transformation techniques
- Developed business-focused analytical thinking
